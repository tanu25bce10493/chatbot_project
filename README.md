# AI Student Assistant Chatbot

## 📌 Overview

This project is an AI-powered chatbot designed to assist students with academic queries using Natural Language Processing and Machine Learning.

## 🚀 Features

* NLP-based intent recognition
* Machine Learning classification
* GUI-based interaction
* Expandable dataset

## 🛠️ Technologies Used

* Python
* Scikit-learn
* Tkinter

## ▶️ How to Run

1. Install dependencies:
   pip install scikit-learn

2. Train model:
   python train.py

3. Run chatbot:
   python app.py

## 📊 Example

Input: What is AI?
Output: AI is the simulation of human intelligence in machines.
